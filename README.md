# PlayZoneX.xyz

PlayZoneX is a GitHub Pages-ready static site for online game discovery, Roblox guides, Minecraft content, Steam recommendations, Blooket, Wordle, gear recommendations, and SEO landing pages.

## Commands

```bash
npm run build
npm run serve
```

`npm run build` generates all static pages into `dist/` and mirrors the deployable static output to the repository root for GitHub Pages.

## Pages

- `/`
- `/online-games/` plus 8 category pages
- `/roblox/` plus 6 guide pages
- `/minecraft/` plus 5 content pages
- `/steam/` plus free games and deals
- `/blooket/`
- `/wordle/`
- `/gear/`
- `/about/`
- `/game/cyber-rage-phantom/`

## Data

Seed content lives in `src/data/site.js`. Generated JSON files are published under `data/` for lightweight client-side usage and future API sync tasks.
